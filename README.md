Rest API quản lý nguời dùng với các chức năng:
- Đăng kí, đăng nhập (JWT)
- Phân quyền ADMIN, USER
- Xóa mềm người dùng
- Xem thông tin chi tiết toàn bộ và thông tin chi tiết một người dùng
- Tìm kiếm thông tin theo id, tên và email
- Cập nhật thông tin người dùng
Công nghệ:
- Visual Studio Code
- Java 17
- Spring Boot
- PostgreSQL
- Postman
- Lombok
- Docker
- Docker Hub image:(https://hub.docker.com/repository/docker/nguyenhuonggiang/baitestfresher-app)

Cài Đặt & Chạy Ứng Dụng
-  [Docker](https://www.docker.com)
-  [Git](https://git-scm.com)
Build & chạy service
docker-compose up --build
